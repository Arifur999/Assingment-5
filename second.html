Question-1: What are the different ways to select an element in the DOM?
DOM (Document Object Model) এটি সাধারণত ৫ রকম ভাবে ডিক্লেয়ার করা যায়
1 - getElementById যাতে শুধু ইউনিক এলিমেন্ট কল করাতে ব্যবহার করা হয়।
2 - getElementsByClassName কে একাধিক এলিমেন্ট কল করার জন্য ব্যবহার করা হয়।
3 - get ElementsByTagName নির্দিষ্ট ট্যাগের এলিমেন্টকে কল করতে ব্যবহার করা হয়।
4 - querySelector প্রথম সিলেক্টরকে কল করতে ব্যবহৃত হয়
5 - querySelectorAll একাধিক CSS query selector কে কল করতে ব্যবহৃত হয়
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML, innerText, এবং textContent—এই তিনটি প্রপার্টি
1. innerHTML এটি কোনো element-এর মধ্যে HTML কোড নতুন করে লেখাটা সরাসরি HTML ফাইলে পাঠানো যায়
2. innerText এটি ব্যবহার করে কোন string এর মান যাচাই করা হয় এবং কোন নাম্বার যদি string হয় তাহলে তাকেও নাম্বারে কনভার্ট করা যায়
3. textContent এটি innerText-এর মতো কাজ করে, তবে HTML ট্যাগ থাকলেও সেটিকে উপেক্ষা করে এবং পুরো টেক্সট কে দেখায়
Question-3: What is event delegation in the DOM?
Event Delegation হলো একটি JavaScript একটি প্রক্রিয়া যেখানে একটি  parent এলিমেন্টে তার ভেতরের যেসব  child  এলিমেন্টে থাকে তাদেরকে কন্ট্রোল করা যায় প্রতিটি child কে আলাদা আলাদা না ধরে parent কে ধরে কাজ করা যায় এতে অনেক সময় বেঁচে যায় এবং বেশি কোড লেখতে হয় না।
Question-4: What is event bubbling in the DOM?
Event Bubbling এর মাধ্যমে কোনো child এলিমেন্টে ইভেন্ট ঘটলে, সেটি তার parent এবং তার উপরের সমস্ত parent দিকে ছড়িয়ে পড়ে। এটি মূলত নিচ থেকে উপরে এবং উপর থেকে নিচে পরিচালিত হয় বলে এটিকে  Bubbling বলে।
Question-5: How do you create, add, and remove elements using JavaScript?
1.নতুন এলিমেন্ট তৈরি: createElement()
2.নতুন এলিমেন্ট যোগ: appendChild(), prepend()
3. এলিমেন্ট মুছে ফেলা: removeChild(), remove()
4.এলিমেন্ট পরিবর্তন: replaceChild()
